# Acesso a Dados Abertos da UFPB

Este projeto visa facilitar o acesso aos dados abertos da Universidade Federal da Paraíba (UFPB). Atualmente, o portal oficial [dadosabertos.ufpb.br](https://dadosabertos.ufpb.br/) está fora do ar, tornando a API a única forma de acesso disponível.

Os dados disponibilizados aqui estão em domínio público, conforme a política de dados abertos do governo e foram obtidos legalmente. 

## Acesso via API

A descrição da API é acessível através do [Wayback Machine](https://web.archive.org/web/20210903155306/https://docs.ckan.org/en/ckan-2.7.3/api/#api-examples), que preserva uma cópia do guia de uso da API conforme publicado anteriormente.

## Formatação de Dados

Foram realizadas correções de formatação nos arquivos CSV disponibilizados por este projeto. Importante ressaltar que tais ajustes não alteram os valores dos dados originais.
